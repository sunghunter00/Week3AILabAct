activity Question 1:

Rule-based systems need override conditions because some rules are more
important than others. In this activity, a teacher's pass should always be
checked first because it gives automatic access to the fast lane. If Aliah
checks the time and number of items first, the program might reject a student
who should actually be approved. Checking the override condition first helps
the program follow the correct rules and avoids incorrect results.

Activity Question 2:

The overdue books check should come first because students with overdue books
are not allowed to borrow no matter what their account status is. If Aliah
checks the student's status first, the program might approve an active student
even if they still have overdue books. Checking the overdue condition first
makes sure the library rules are followed correctly and prevents incorrect
borrowing approvals.
